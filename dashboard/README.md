# BedemWAF Dashboard

The dashboard is the TypeScript and Next.js admin UI for BedemWAF.

Planned responsibilities:

- Manage applications, origins, policies, rule groups, IP sets, and rate limits
- Display WAF events and analytics
- Support safe count-to-block rollout workflows
- Apply secure browser headers and avoid exposing secrets client-side

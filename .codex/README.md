# Codex Workspace

This directory contains repo-local context for Codex agents working on
BedemWAF.

Files:

- `project.md`: product architecture, service ownership, and implementation
  boundaries.
- `commands.md`: common commands with cache-safe variants for this environment.
- `backlog.md`: safe next tasks grouped by service.

The root `AGENTS.md` is the primary entrypoint. Keep this directory focused and
short so future agents can load it quickly.
